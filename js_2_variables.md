# 變數

聲明一個名稱並儲存值
可以在同一行聲明多個變數

```js {.line-numbers}
let user = 'John', age = 25, message = 'Hello';
```

分行聲明 (注意符號)

```js {.line-numbers}
let user = 'John',
    age = 25,
    message = 'Hello';
```

## 變數命名

必須是字母 數字 符號 $ 和 _ ,且字首不是數字

---

## 常數

就是一個不能變動的變數

```js {.line-numbers}
const user = 'John' ;
user = 'Eric' // 錯誤
```
